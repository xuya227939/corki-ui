## Install

```bash
npm install corki-ui --save
```

```bash
yarn add corki-ui
```

## Example

````jsx
import { Button } from 'corki-ui';

const App = () => (
  <>
    <Button type="primary">PRESS ME</Button>
  </>
);
````

## Get started

```bash
$ git clone https://github.com/xuya227939/corki-ui.git
$ cd corki-ui
$ npm install
$ npm start
```
